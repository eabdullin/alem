## Qurt v0.1.1

### What's new

- **Checkpoint restore** — Restoring conversations to a previous point is now more reliable and faster
- **File-read tool** — The agent can now read files from your machine during a conversation
- **Chat UI improvements** — Cleaner message layout, better tool step display, and various visual fixes

### Downloads

| Platform | File | Instructions |
|----------|------|--------------|
| **Windows** | `Qurt-v0.1.1-Windows-Setup.exe` | Run the installer and follow the setup wizard |
| **macOS** | `Qurt-v0.1.1-macOS.dmg` | Open the DMG and drag Qurt to Applications |
